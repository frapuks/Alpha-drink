# ALPHA DRINK

Application privée pour la gestion d'une carte de boissons.

[=> Sommaire du projet](Docs/0-Sommaire.md)

# Technologies utilisées

- Node
- Express
- Express-session
- PostgreSQL
- Ejs
- Bcrypt
- CSS
- Javascript Vanilla
